## 快速概览

### 项目定位

**Clawme（虾米）** 是一个以"动态大盘 (Feeds)"与"协作会话 (Chat)"为双核心的个人本地数字助理与知识归档引擎。

### 技术栈

- **前端**: Nuxt 3 + @nuxt/ui
- **数据库**: Prisma + PostgreSQL (JSONB)
- **AI 引擎**: Ollama/LM Studio/oMLX + MCP + Workflows

### 开发阶段

1. **Phase 1**: 核心工具底座与 UI 骨架
2. **Phase 2**: 画廊瀑布流、流式对话与记录导入
3. **Phase 3**: 群聊会话、MCP 协议栈与工作流引擎
4. **Phase 4**: 赛博生态引擎注入
5. **Phase 5**: 真实生命感与社交图谱
