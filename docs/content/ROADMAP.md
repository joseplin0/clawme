# 当前路线图

本文件是当前执行计划的唯一入口，只描述基于当前仓库现实的执行路线，不再把远期设想和近期任务混在一起。

## 已落地基线

- Nuxt 4 + `@nuxt/ui` 应用壳已完成
- Drizzle + PostgreSQL 数据层已完成
- 首次引导、管理员登录与基础鉴权已完成
- Feed 页面与设置页已完成基础可用版本
- 会话列表、会话详情、HTTP 流式聊天、WebSocket 实时聊天已接通

## P0：近期必须补齐

- 接通聊天页“创建会话”交互，前端真正使用 `POST /api/chat/session/index`
- 为 `useActors()` 补 `/api/actors/:id`，或者移除这条未完成依赖
- 明确 `server/api/agent.post.ts` 的去留，避免新协作者误判主链路
- 补一份部署/运行文档，说明数据库准备、初始化流程和最小环境要求

## P1：下一个里程碑

- 实现 ChatGPT / Gemini 导入器
- 实现面向外部 Agent 或 BOT 的 Bearer Token 写入 API
- 为聊天主链路增加最小测试或 smoke check
- 收敛系统配置、Provider 管理和后续运营配置文档

## P2：中期扩展

- 群聊会话与多参与者协作
- MCP 工具接入
- 工作流编排
- Feed 更完整的互动能力，例如评论、详情、草稿转发

## 暂缓事项

在基础协作、导入和接入能力没补齐之前，不建议优先投入以下方向：

- 完整赛博生态调度

- 大规模“生命感”玩法扩展
