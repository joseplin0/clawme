## Why

当前聊天界面组件（ChatBox.vue、ChatList.vue）的样式较为基础，布局偏向传统应用风格。为了提升用户体验，需要将其改造为更接近现代聊天软件的交互模式：左侧会话列表顶部提供快速新建入口，右侧消息区顶部显示当前对话信息和操作按钮。

## What Changes

- **ChatList.vue（左侧会话列表）**:
  - **移除原标题区域**，替换为搜索/输入框 + 新建按钮（+）
  - 输入框支持快速搜索/筛选会话
  - 新建按钮用于创建新会话
  - 添加会话卡片的悬停效果和选中状态样式
  - 优化会话列表项的间距和圆角设计
  - 增加选中会话的高亮边框或背景效果

- **ChatBox.vue（右侧消息区）**:
  - **Header 区域重构**：
    - 左侧：显示当前对话标题
    - 右侧：操作按钮组（如 Feed Draft、设置等）
  - 优化消息气泡的视觉区分（用户/AI 消息样式）
  - 改进输入框区域的视觉层次
  - 增加空状态页面的视觉设计

- **通用样式**:
  - 统一颜色变量和间距规范
  - 添加适当的阴影和过渡动画
  - 优化响应式布局下的样式表现

## Capabilities

### New Capabilities

- `chat-ui-theming`: 聊天界面的主题化样式系统，包含会话列表样式、消息气泡样式、输入框样式等视觉规范

### Modified Capabilities

无。这是一个纯 UI 样式改进，不涉及现有功能的行为变更。

## Impact

- **受影响文件**:
  - `app/components/chat/ChatBox.vue`
  - `app/components/chat/ChatList.vue`
  - 可能新增 `app/assets/css/chat.css` 样式文件

- **依赖**: 无新增依赖，使用现有的 Nuxt UI 组件和 Tailwind CSS

- **向后兼容**: 完全兼容，仅为样式和布局增强
